# 安装

YCloud Icons 当前覆盖 10 种包类型，可以按项目所用框架选择安装。

## Core JavaScript

与框架无关的图标数据和辅助方法。

::: code-group

```sh [pnpm]
pnpm add @ycloud-web/icons
```

```sh [npm]
npm install @ycloud-web/icons
```

```sh [yarn]
yarn add @ycloud-web/icons
```

```sh [bun]
bun add @ycloud-web/icons
```

:::

查看 [Core JavaScript 指南](./ycloud/index.md)。
