# YCloud Static

This package contains the following YCloud Icons static asset implementations:

- All SVG files
- A JavaScript library containing SVG strings
- Icon Font
- SVG sprite

> Want to learn about YCloud Icons? Read the [project introduction](https://github.com/TianJianJun0727/ycloud-icons#ycloud-icons).

## Why use @ycloud-web/icons-static?

This package is for very specific static asset scenarios, such as using Icon Font, SVG sprite, plain SVG files, or CommonJS SVG strings in JavaScript projects.

> [!WARNING]
> Unless you specifically need static assets, SVG sprite or Icon Font should not be the first choice for production web applications. Prefer the framework package for your stack so the bundler can tree-shake unused icons: `@ycloud-web/icons`, `@ycloud-web/icons-react`, `@ycloud-web/icons-vue`, `@ycloud-web/icons-svelte`, `@ycloud-web/icons-solid`, `@ycloud-web/icons-preact`, `@ycloud-web/icons-react-native`, `@ycloud-web/icons-angular`, and `@ycloud-web/icons-astro`.

## Installation

```sh
pnpm add @ycloud-web/icons-static@latest
```

```sh
npm install @ycloud-web/icons-static@latest
```

```sh
yarn add @ycloud-web/icons-static@latest
```

```sh
bun add @ycloud-web/icons-static@latest
```

### Version requirements

No framework peer dependency.
