# YCloud Solid

Implementation of the YCloud icon library for solid applications.

## Installation

```sh
pnpm add @ycloud-web/icons-solid
```

```sh
npm install @ycloud-web/icons-solid
```

```sh
yarn add @ycloud-web/icons-solid
```

```sh
bun add @ycloud-web/icons-solid
```
