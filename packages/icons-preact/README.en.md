# YCloud Preact

YCloud Icons implementation for Preact applications.

## Installation

```sh
pnpm add @ycloud-web/icons-preact@latest
```

```sh
npm install @ycloud-web/icons-preact@latest
```

```sh
yarn add @ycloud-web/icons-preact@latest
```

```sh
bun add @ycloud-web/icons-preact@latest
```

### Version requirements

Preact `^10.27.2`.

## Documentation

Read the full documentation at [YCloud Icons docs](https://tianjianjun0727.github.io/ycloud-icons/en/guide/preact).
