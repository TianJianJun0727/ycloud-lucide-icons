# YCloud Solid

YCloud Icons implementation for Solid applications.

## Installation

```sh
pnpm add @ycloud-web/icons-solid@latest
```

```sh
npm install @ycloud-web/icons-solid@latest
```

```sh
yarn add @ycloud-web/icons-solid@latest
```

```sh
bun add @ycloud-web/icons-solid@latest
```

### Version requirements

Solid `^1.4.7`.

## Documentation

Read the full documentation at [YCloud Icons docs](https://tianjianjun0727.github.io/ycloud-icons/en/guide/solid).
