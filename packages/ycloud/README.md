# YCloud

Implementation of the YCloud icon library for web applications.

## Installation

```sh
pnpm add ycloud
```

```sh
npm install ycloud
```

```sh
yarn add ycloud
```

```sh
bun add ycloud
```

### CDN

```html
<!-- Development version -->
<script src="https://unpkg.com/ycloud@latest/dist/umd/ycloud.js"></script>

<!-- Production version -->
<script src="https://unpkg.com/ycloud@latest"></script>
```
