# YCloud Svelte

YCloud Icons implementation for Svelte applications.

## Installation

```sh
pnpm add @ycloud-web/icons-svelte@latest
```

```sh
npm install @ycloud-web/icons-svelte@latest
```

```sh
yarn add @ycloud-web/icons-svelte@latest
```

```sh
bun add @ycloud-web/icons-svelte@latest
```

> `@ycloud-web/icons-svelte` supports only Svelte 5; Svelte 4 is not supported in the current version.

### Version requirements

Svelte `^5`. This package supports only Svelte 5; Svelte 4 is not supported in the current version.

## Documentation

Read the full documentation at [YCloud Icons docs](https://tianjianjun0727.github.io/ycloud-icons/en/guide/svelte).
