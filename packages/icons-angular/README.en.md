# YCloud Angular

YCloud Icons icon library for Angular applications, supporting standalone components, Signals, and zoneless usage.

## Installation

```sh
pnpm add @ycloud-web/icons-angular@latest
```

```sh
npm install @ycloud-web/icons-angular@latest
```

```sh
yarn add @ycloud-web/icons-angular@latest
```

```sh
bun add @ycloud-web/icons-angular@latest
```

### Version requirements

Angular `>=17.0.0`.

## Documentation

Read the full documentation at [YCloud Icons docs](https://tianjianjun0727.github.io/ycloud-icons/en/guide/angular).
