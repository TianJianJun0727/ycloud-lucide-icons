# YCloud Astro

Implementation of the YCloud icon library for Astro applications.

## Installation

```sh
pnpm add @ycloud-web/icons-astro
```

```sh
npm install @ycloud-web/icons-astro
```

```sh
yarn add @ycloud-web/icons-astro
```

```sh
bun add @ycloud-web/icons-astro
```
