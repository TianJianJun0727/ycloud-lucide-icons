# YCloud Icons Vue

YCloud Icons implementation for Vue applications.

## Installation

```sh
pnpm add @ycloud-web/icons-vue@latest
```

```sh
npm install @ycloud-web/icons-vue@latest
```

```sh
yarn add @ycloud-web/icons-vue@latest
```

```sh
bun add @ycloud-web/icons-vue@latest
```

### Version requirements

Vue `>=3.0.1`.

## Documentation

Read the full documentation at [YCloud Icons docs](https://tianjianjun0727.github.io/ycloud-icons/en/guide/vue/).
